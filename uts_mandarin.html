UTS Mandarin
Harvesting System

Simulated a fruit harvesting robotic system which managed two 6-DOF robots to pick and sort mandarins. This program was written in MatLab and utilised the App Designer to promote end-user UI and UX.


Project Introduction

This group project stemmed from a UTS Industrial Robotics assignment where we had free reign to choose a project to solve a real world problem.
We chose to address the labour shortages in the Australian farming industry due to the Covid pandemic.

Project Scope

- Build a simulated model of two robots and environment for a chosen scenario. In this case, a fruit harvesting system.
- Include a MatLab user interface (GUI) to interact with the system. The GUI should control the systemâ€™s functionality including movement of the robots.
- The system must have a functional emergency stop button.
- The system must prevent and avoid collisions.
